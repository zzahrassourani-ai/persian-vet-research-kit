# Persian Vet Research Kit 🐾🇮🇷

مجموعه کامل منابع و ابزارهای تحقیقاتی برای دانشجویان و محققان دامپزشکی ایران.

این پروژه با هدف کمک به دانشجویانی ساخته شده که دسترسی به ابزارهای گران‌قیمت ندارند و می‌خواهند مقالات علمی با کیفیت بالاتری بنویسند.

## محتوای پروژه

- **تمپلیت‌های مقاله‌نویسی** (فارسی و انگلیسی)
- **چک‌لیست‌های تحقیقاتی**
- **اصطلاح‌نامه تخصصی دامپزشکی**
- **راهنماهای روش تحقیق**
- **لیست منابع و نرم‌افزارهای رایگان**

## چگونه استفاده کنیم؟
هر فولدر را باز کنید و فایل‌های مورد نیاز را دانلود کنید.

## هدف پروژه
کمک به افزایش کیفیت تحقیقات دامپزشکی در ایران و کشورهای در حال توسعه.

## Contributing
اگر دانشجو یا محقق دامپزشکی هستید، خیلی خوشحال می‌شویم فایل‌ها یا تمپلیت‌های خودتان را贡献 کنید.
